# Contributing to Smart-Check

Thanks for your interest in contributing! Here's how you can help.

## Getting Started

1. Fork the repository
2. Clone your fork: `git clone https://github.com/yourusername/smart-check.git`
3. Create a branch: `git checkout -b feature/your-feature-name`
4. Make your changes
5. Test locally
6. Commit: `git commit -m "Add: your feature description"`
7. Push: `git push origin feature/your-feature-name`
8. Open a Pull Request

## Development Setup

```bash
# Install frontend dependencies
cd web
npm install
npm run dev

# Install backend dependencies
cd backend
npm install
```

## Code Style

- Use ESLint for JavaScript/React
- Follow existing code patterns
- Write meaningful commit messages
- Add comments for complex logic

## Testing

Before submitting a PR:
- Test all features manually
- Ensure no console errors
- Test on mobile viewport
- Verify QR scanning works

## Feature Ideas

Check the roadmap in README.md for planned features. Some good first contributions:
- Add unit tests
- Improve mobile UI
- Add dark mode
- Implement search/filter
- Add export to CSV

## Questions?

Open an issue or reach out via GitHub Discussions.
